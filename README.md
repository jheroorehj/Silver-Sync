# Silver-Sync

고령 환자 만성질환 관리를 위한 AI 기반 의료 협업 플랫폼 데모입니다.

## 프로젝트 구조

| 폴더 | 설명 |
|------|------|
| `FE_DEMO/` | Silver-Sync 프론트엔드 데모 (의사/간호사/환자 뷰) |
| `genticsimlaw-flowchart/` | 유전 심폐소생술 법 RAG 플로우차트 시각화 |

## 실행 방법

각 폴더로 이동 후 아래 명령어를 실행합니다.

```bash
npm install
npm run dev
```

자세한 내용은 각 폴더의 README.md를 참고하세요.
